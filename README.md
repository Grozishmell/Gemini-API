# Gemini-pdf-bot : вопросы к документу на базе RAG

Приложение создавалось для использования Gemini-API вместе с pdf-файлом в веб-интерфейсе

Веб-приложение, которое отвечает на вопросы по содержимому загруженного PDF.
Построено на технологии **RAG** (Retrieval-Augmented Generation) с использованием
моделей Google Gemini.

Загрузите PDF, задайте вопрос обычными словами — и бот найдёт ответ
в документе и сформулирует его человеческим языком. Если ответа в документе
нет, бот честно об этом сообщит, а не выдумает.

## Возможности
 
- Загрузка любого PDF через интерфейс
- Вопросы на естественном языке
- Поиск по **смыслу**, а не по точному совпадению слов
- Ответы строго на основе документа (защита от «галлюцинаций»)
- Простой веб-интерфейс, работает в браузере

## Технологии
 
- **Python**
- **Streamlit** — веб-интерфейс
- **Google Gemini API** — генерация ответов (`gemini-3.5-flash`) и эмбеддинги (`gemini-embedding-001`)
- **pypdf** — чтение PDF
- **NumPy** — вычисление близости векторов

## Как это работает
 
Приложение реализует классический пайплайн RAG из пяти шагов:
 
1. **Чтение** — из PDF извлекается весь текст.
2. **Нарезка** — текст делится на небольшие пересекающиеся кусочки (чанки).
3. **Эмбеддинги** — каждый кусочек превращается в вектор, отражающий его смысл.
4. **Поиск** — вопрос тоже превращается в вектор, и по косинусной близости находятся
   наиболее релевантные кусочки документа.
5. **Генерация** — найденные кусочки передаются модели как контекст, и она формулирует
   ответ, опираясь только на них.
Эмбеддинги документа вычисляются один раз при загрузке и кэшируются в памяти сессии,
чтобы не пересчитывать их при каждом вопросе.

## Установка
 
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/pdf-rag-bot.git
cd pdf-rag-bot
 
# 2. Установите зависимости
pip install -r requirements.txt
```
 
## Настройка API-ключа
 
1. Получите бесплатный ключ в [Google AI Studio](https://aistudio.google.com).
2. Создайте файл `.streamlit/secrets.toml` и добавьте в него ключ:
```toml
GEMINI_API_KEY = "ваш_ключ"
```
 
> Файл с ключом не попадёт в репозиторий — он указан в `.gitignore`.
 
## Запуск
 
```bash
streamlit run pdf_bot.py
```
 
Приложение откроется в браузере. Загрузите PDF и задавайте вопросы.
